# Exercise 3: Mobile-First Design

Implementation of the layout optimized for mobile devices, prioritizing vertical flow and usability on small screens.

## Features
* **Mobile-First Approach:** Layout designed to be fully functional on small viewports.
* **Flexbox Layout:** Utilizes `flex-direction: column` for a clean, stacked vertical hierarchy.
* **Width Check Utility:** Includes a JavaScript-powered button to verify the current `window.innerWidth` during testing.
