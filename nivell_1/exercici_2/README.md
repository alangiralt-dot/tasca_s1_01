# Exercise 2: Tablet Adaptation

This stage focuses on tablet responsiveness by introducing breakpoints and more complex layout structures.

## Features
* **Media Query:** Breakpoint set at `700px` to handle tablet-sized viewports.
* **CSS Grid:** Advanced distribution of the article section using `grid-template-columns` and `grid-template-rows`.
* **Structural Flexibility:** Uses a combination of Flexbox for the main containers and Grid for internal content.
